# LISEZ-Moi

Journal en ligne indépendant - Comprendre avant de juger

## Déploiement

### Frontend (GitHub Pages)
Le frontend se déploie automatiquement via GitHub Pages.

### Backend (Render)
1. Créer un Web Service sur Render
2. Root Directory: `backend`
3. Build Command: `npm install`
4. Start Command: `npm start`

### Variables d'environnement (Render)
- `MONGODB_URI` : URI MongoDB Atlas
- `PORT` : 3000

## Développement local
```bash
cd backend
npm install
npm start
```
